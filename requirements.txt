1.Install Python - sudo apt-get install python3.6.
2.Install VScode IDE - sudo snap install --classic code.
3.make directory - mkdir.
4.Inside the directory, install VirtualEnv - sudo apt-get install python3-venv.
5.Name the VirtualEnv - python3 -m venv pyvenv  ->filename).
6.Activte the virtualEnv for directory - source pyvenv/bin/activate.
7.Install pip package for python3 - sudo apt install python3-pip.
8.Install Django - pip install django.
9.Create the MainProject inside directory - django-admin startproject HRmanagement .
10.Create the actual project - python manage.py startapp HRMS.
11.Inside the actual project create templates folder, inside it create a subfolder with actual project name to store html files.
12.For db connection install a drive for Mysql - sudo apt-get install python3-dev default-libmysqlclient-dev build-essential.
13.Install mysql - pip3 install mysqlclient.
14.Change the db config on setting.py file.
15.Install rest framework - pip3 install djangorestframework.
